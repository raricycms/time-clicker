🕰️ 时光机修复计划 - 游戏启动说明

📋 快速开始：
1. 双击 index.html 文件
2. 游戏将在默认浏览器中打开
3. 开始点击时光机收集时间能量！

🎮 游戏操作：
- 点击时光机：收集时间能量
- 购买建筑：自动产生能量
- 空格键：快速点击时光机
- 左上角菜单：查看统计、设置等

🎯 游戏目标：
收集1万亿点时间能量来修复时光机！

🏗️ 建筑系统：
- 时间沙漏 → 时钟工厂 → 时间加速器 → 时空裂缝 → 时间机器原型
- 时间循环器 → 平行宇宙探测器 → 时间悖论引擎 → 时空稳定器 → 终极时光机

🌌 特殊事件：
游戏会随机触发时空事件，可能带来正面或负面影响

🏆 成就系统：
完成特定目标解锁成就，获得成就感！

💾 存档系统：
- 游戏自动保存
- 支持手动导出/导入存档
- Ctrl+S 手动保存

⚙️ 设置选项：
- 自动保存间隔
- 事件触发概率
- 音效开关
- 动画效果开关

🔧 技术信息：
- 纯HTML5 + CSS3 + JavaScript
- 无需安装，直接在浏览器中运行
- 支持现代浏览器（Chrome、Firefox、Safari、Edge）

📱 兼容性：
- 桌面浏览器：完全支持
- 移动设备：基本支持（触摸点击）

🎨 游戏特色：
- 科幻风格设计
- 动态光效和动画
- 响应式界面
- 丰富的游戏机制

开始你的时空修复之旅吧！🚀
